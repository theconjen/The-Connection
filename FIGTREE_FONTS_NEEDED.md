# Figtree Fonts Required

## Download Instructions

1. Go to [Google Fonts - Figtree](https://fonts.google.com/specimen/Figtree)
2. Click "Download family"
3. Extract the ZIP file
4. Copy these files to `/assets/fonts/`:
   - `Figtree-Regular.ttf` (from static folder)
   - `Figtree-Medium.ttf` (from static folder)
   - `Figtree-SemiBold.ttf` (from static folder)
   - `Figtree-Bold.ttf` (from static folder)

## Direct Download Links (if available)

Alternatively, you can download individual weights from:
- https://github.com/erikdkennedy/figtree/tree/main/fonts/ttf

## Required Files

Place these in `/assets/fonts/`:
```
assets/fonts/
├── Figtree-Regular.ttf
├── Figtree-Medium.ttf
├── Figtree-SemiBold.ttf
└── Figtree-Bold.ttf
```

After adding fonts, the app will load them automatically on startup.
